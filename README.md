## Análise de Dados e Manipulação com Python
Este projeto demonstra a utilização de bibliotecas essenciais do Python para manipulação, análise e visualização de dados, especificamente NumPy, Pandas e Matplotlib.

Conteúdo do Notebook:
Manipulação de Arrays com NumPy:

Criação de arrays 1D (unidimensional) e 2D (bidimensional).
Cálculo da média (np.mean()) de ambos os arrays.
Cálculo da soma (np.sum()) dos elementos de ambos os arrays.
Análise de Dados com Pandas:

Carregamento de um dataset público (iris.csv) diretamente de uma URL (https://raw.githubusercontent.com/mwaskom/seaborn-data/master/iris.csv) para um DataFrame do Pandas.
Exploração inicial do DataFrame usando métodos como df.info(), df.describe(), df.head() e df.tail() para entender a estrutura, tipos de dados e estatísticas descritivas.
Visualização de Dados com Matplotlib:

Criação de um gráfico de dispersão (plt.scatter()) para visualizar a relação entre as colunas 'sepal_length' e 'sepal_width' do dataset Iris.
Geração de um histograma (plt.hist()) para mostrar a distribuição da coluna 'petal_length'.
